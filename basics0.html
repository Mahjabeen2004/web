WEB APPLICATION

A web application is an application programme that is started on a remote server and delivered over the internet through a browser interface.

WEB APPLICATION COMPONENTS

  1. User interface component
  2. Structure component
  3. Client component
  4. Server component

BENIFITS OF WEB APPLICATION

  • Allowing multiple users access to the same version of an application
  • Web apps don't need to be installed
  • Can be accessed through multiple users
